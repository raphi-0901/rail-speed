# rail-speed gnome extension

## Local Development

Reenable extension:
```bash
gnome-extensions reset rail-speed@raphael-wirnsberger.at ; gnome-extensions enable rail-speed@raphael-wirnsberger.at
```

Reload extension:
`ALT` + `F2`, then `r` and `enter`

### See logs
```bash
journalctl -f -o cat | grep rail-speed
```
